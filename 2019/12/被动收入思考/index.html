被动收入思考

十二月 21, 2019 发布在 2019

想F.I.R.E，但资金还是差了那么一丢丢，为了应对通膨，需要最基本的被动收入来支撑。

于是整理了一些对被动收入的理解和分析…

但是，依然没有什么太多的收获，作为以后备忘参考吧。

概念

特点：一份时间可以卖N次

重点：被动收入是吃苍蝇肉，要有耐心，要积少成多

技巧：

想办法打组合拳，让若干方式能互相支撑起来

例如写文章挂小广告，打出声誉，然后可以做收费课程，整理出书出售，做收费顾问咨询。

要刻意养成找到那种一次付出，却刻意长期收入的事情去做

例如程序打工就很差，程序技能本身也属于差的投资，而医生，厨师，写作都算很稳定的技能。

方式记录

  1. 以财理财（核心在于投资组合）
    • 储蓄利息
    • 基金
    • 保险
    • 信托
    • 债券
    • 股票
    • 创建一个投资组合
      • 成功概率：★★★★★ 世界上最简单的方法就是…假如你有钱
      • 金钱开销：★★★★★
      • 花费时间：★ 简单选择被动型投资，并不花费时间
      • 回报情况：★★ 年化一般是5-10%，其中2-3%的通膨
      • 收益预估：
      • 核心要点：注意，尽量有更多余地，以便将剩余的收入进行再投资，产生复利效果。
      • 推荐关键字：首要推荐ETF，其次推荐指数基金。
    • 社交借贷
      • 成功概率：★★★
      • 金钱开销：★★
      • 花费时间：★
      • 回报情况：★
      • 收益预估：
      • 核心要点：国外很流行非常小额的认识的人之间的借贷，可以获得比银行利率稍高的报仇
      • 推荐关键字：
      • 倒货
      • 成功概率：★★
      • 金钱开销：★★★
      • 花费时间：★★★
      • 回报情况：★★
      • 收益预估：
      • 核心要点：从海外大量批货，在网上进行出售或拍卖
      • 推荐关键字： 要有很强的商品选择能力
  2. 房产
    • 房租
      • 成功概率：★★★★ 风险来自于可能的高杠杆
      • 金钱开销：★★★★★ 通常要求资本很高，且可能需要高杠杆
      • 花费时间：★★ 需要学习基本的相关知识
      • 回报情况：★★★ 扣除物业管理，收入其实不算太高
      • 收益预估：
      • 核心要点：如果没有大资金，就需要高杠杆。如果想被动收入，就需要物业管理人。
      • 推荐关键字：
    • AirBnb短租
    • 转租
  3. 知识产权
    • 专利使用费
    • 顾问咨询收费
    • 文章稿费（写网络小说）
    • 写电子书出售
      • 成功概率：★★ 中国人不爱付费读电子书，国外人读书但我们写不出来外语的
      • 金钱开销：★ 基本不花钱，买点书看看开拓思路就好
      • 花费时间：★★★ 比较花费时间
      • 回报情况：★★ 收入不高，但后期几乎完全无成本
      • 收益预估：月收1000
      • 核心要点：做好SEO工作，推荐考虑再包装方式，例如先写博客公众号，在将其包装整理为电子书，线上课程等。
      • 推荐关键字：百度阅读，网易云阅读，iBOOK Author，Kindle，GitNook
    • 付费文章
      • 微信公众号，简书，知乎，豆瓣，今日头条等
      • 建议写非时效性的，不可替代性的文章
    • 视频课程培训收费（Udemy，网易云课堂，荔枝微课等）
      • 成功概率：★★★ 主要困难依然是SEO和内容部分
      • 金钱开销：★ 前期会有一点消耗，包括麦克风摄像头等工具
      • 花费时间：★★★★ 比较花费时间，估计比Blog还要消耗多一点
      • 回报情况：★★★★ 收益其实还可以。
      • 收益预估：月8000
      • 核心要点：很累很麻烦，收益率倒是挺高
      • 推荐关键字：
    • 摄影/画作版权
      • 成功概率：★ 免费图库已经很多了，必须选好方向
      • 金钱开销：★★★★
      • 花费时间：★★★
      • 回报情况：★★★ 如果成功了，收入还是会挺不错的
      • 收益预估：
      • 核心要点：如何获取大量的素材，如何和免费图库竞争
      • 推荐关键字：
    • 设计创意物件
      • 成功概率：★ 行销非常重要，不然将默默无闻
      • 金钱开销：★
      • 花费时间：★★
      • 回报情况：★★ 竞争十分激烈
      • 收益预估：
      • 核心要点：例如马克杯，T-shirt，钥匙链，公仔等等。成本低，但关键在于大量。例如10种产品 * 100张照片 = 1000种产品
      • 推荐关键字：
  4. 互联网类

        • 金钱开销：★ 基本不花钱，域名加主机一年不用1000
        • 花费时间：★★★★ 比较花费时间，需要长期经营
        • 回报情况：★★★ 收入一般，比较稳定
        • 收益预估：月收1500
        • 核心要点：必须做好专题方向和内容，做好流量的精准度。做好SEO工作。
        • 推荐关键字：WordPress, GoogleAdSense, Amazon联盟营销计划，WooCommerce电子商务系统
      • 比较评论站
        • 成功概率：★★
        • 金钱开销：★★★ 要购买不少的产品做测试
        • 花费时间：★★★★★ 基本要全身心投入
        • 回报情况：★★★ 收入还不错，有概率成为大站
        • 收益预估：
        • 核心要点：
        • 推荐关键字：
      • 高UV的niche专项网站
        • 成功概率：★★★
        • 金钱开销：★★
        • 花费时间：★★★★ 编写内容，找厂商沟通都很重要
        • 回报情况：★★★ 市场规模很重要
        • 收益预估：月收8000

      • 高niche的组群交流和订阅频道
        • 成功概率：★
        • 金钱开销：★★★
        • 花费时间：★★★★

        • 收益预估：

        • 金钱开销：★★★★

        • 收益预估：
        • 核心要点：因为找产品，找工厂，所以前期推荐流程特别繁琐

        • 金钱开销：★★

        • 收益预估：

        • 推荐关键字：

        • 金钱开销：★
        • 花费时间：★★
        • 回报情况：★

        • 核心要点：靠赞助过日子，不要抱有太大期望

        • 成功概率：★ 很难找到优秀赚钱的项目

        • 回报情况：★
        • 收益预估：
        • 核心要点：不管是折价卷，集点卡，现金回馈等，很简单，但收益很少，一年能有5000块钱就不错了。如果批量甚至自动化的薅羊毛，则可以多一些收入
        • 推荐关键字：

      • 回报情况：★★★★★ 收
      • 收益预估：
      • 核心要点：只要解决法律，管理，沟通的问题，就可以将很多“服务性质”的工作进行外包，并从中收取利润。例如， 写作，程序设计，美术设计，咨询类，法律类，工程类工作都可以外包
      • 推荐关键字：

      • 金钱开销：★★★★
      • 花费时间：★★★★

      • 核心要点：就是自己创造产品，可以在网上出售。这样利润很大，但投入成本和风险都很大，还可能会有存货。
      • 推荐关键字：
    • 企业分红

      • 花费时间：★★

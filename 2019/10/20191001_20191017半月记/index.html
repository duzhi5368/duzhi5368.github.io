我的航海计划

最近公司推荐看书《能力陷阱》，里面很核心的就是提出“要走出自己能力的范围”，其实就是那一套confortable zone的概念。

不禁思考，自己总是沉在程序中，导致第二职业总是发展不起来，移民后会比较难办。所以总在思考如何跳出沉迷程序技术的怪圈。

期间想到自己水肺100瓶也是一种成本投资，其实只要有3个月空闲时间，应该可以达成教练的。自由潜也有了证。能与这两个优势形成技能支点的就是很早之前就想进一步深入的帆船驾驶。

于是进一步了解了RYA和帆船本身。

RYA的学习进阶过程大约是

  • START YACHTING 两天书本课，没太大意义
  • COMPETENT CREW 五天+的船上训练，这个结束就算是船员了
  • DAY SKIPPER 五天+的船上训练，100海里，4小时夜课加书本考试，考过就可以自己租船出去玩了，但只能白天玩
  • COASTAL SKIPPER 15天海训，300海里，8小时夜课。考过就可以稍微远洋一些，并且允许晚上开船了
  • YACHTMASTER COASTAL， YACHTMASTER OFFSHORE， YACHTMASTER OCEAN这些就是可以越跑越远了，我也不具体说了，估计走专业路线了。

就当前来说，能考到COASTAL SKIPPER基本就可以横着走了，除非穿洋环球比赛，这也不是现在考虑的。

价格并不贵，COMPETENT CREW学费10000RMB左右，船上5天比较轻松。下面是马来西亚兰卡威的价格（美元结算）

不过这个学完了不能赚钱，只能做兴趣。所以没必要研究的太深。如果沉迷了就拿个coastal，入门的时候2500刀拿个day skipper就足够了。

另外，帆船和游艇对我而言最重要的其实是价格，没错，太穷的人只能重点考虑这个因素。通常而言，帆船游艇也都是因体长决定价格区间的。

20ft -，那就是代步车，mini yacht只能游泳
20ft - 30ft，约莫50万上下，但是多半没房间，出门晒晒太阳钓个鱼晚上就回来还行，潜水和船宿是没戏的
30ft - 50ft，约莫百万人民币，一般可以多人，5-6人的样子，可以多日居住，可以offshore混，
50ft + ，起码50W刀，不能考虑了

我琢磨自己买船的话，主要还是家庭娱乐，偶尔有概率带几个人去潜潜水或者近海航行旅游，所以什么特殊的钓鱼艇赛艇巡逻艇商务艇运动艇就再见，肯定是休闲艇的。

考虑到停泊位价格，油耗基本和船身大小完全正比的前提下，我觉得最适合的还是32-42ft之间的小龙骨辅助动力帆船（考虑帆船原因是我也想多锻炼锻炼自己，也能出海更远一点）。

这东西不考虑品牌的话，约莫要100W人民币左右。（不过，我看到bilibili里面一个90后大神妹子，居然在ebay上用5000英镑就买了个2手46尺双桅船哦！搞的我从被窝爬起来充电查ebay…的确有很便宜的，但是挺难淘到好的，我在想要不要写个python爬虫盯着它…）

真正麻烦的是泊位，这种中小船在国外良好港口不少的地方约莫一个月也要5000-10000软妹币，所以这事还要多考量考量，或者不得不考虑更多的将它用作商业用途。

现在想得到的只有：

  • 带团短期旅行，什么马尔代夫5日游艇游啦之类的
  • 升级教练，授课带船长（建议ASA船长，比RYA容易）
  • 改装成潜水船，带客人潜水。
  • 将游艇提供给用户拍照，一日游或者船上吃饭之类的。

其实都不太容易，而且看起来并不令人十分愉快。

不过，这些都不妨碍我去考个day skipper试试看先，现在对这个圈子理解还不够。

顺道记录一下，我想学帆船进行帆船旅行的原因：

  • 不想一辈子只做一件事或者从事一个职业（例如仅做程序员）
  • 个人不喜欢和太多的人接触，知己好友两三个已经完全足矣（帆船的容载大小简直完美）
  • 很喜欢探索新东西，感受新鲜的文化思想和生活，想”读万卷书行万里路”（帆船远航算是很不错的一种方式，可以到处歇脚2-3月）
  • 喜欢接触自然，顺应自然，喜欢保持原始敬畏自然的生活方式（总感觉科学文明发展太快，人类肉体和灵魂还没能进化到同步适应的地步）
  • 可能可以有效的躲避极端危机事件（……嗯，我有不正常的一种认知，总觉得全球战争，核辐射，全球性的病毒感染这些不是太远）
  • 本身航行也磨练意志，也相当酷，对吧~哈哈

我不适合进行长期帆船旅行的缺点：

  • 开销比较大，需要有雄厚资产支持，或者想到有效收入的方式（原计划房车freelancer勉强还行，但船上信号问题就不太现实了）
  • 孩子的教育发展问题难以解决。长期出海不在孩子身边肯定不好；而如果长期将孩子带在船上的话，知识教育可能还行，但社交能力就很容易完蛋。
  • 老婆的承受能力。我是野生性格，比较耐磨耐折腾，但我妻子不是，她还是比较主流传统的向往日韩小资家居女生的味道，虽然现在也被带的爱跑动了一些，恐怕长期在船上奔波也是她不能承受的。

一个有趣的妹纸

搜索RYA中无意发现一个叫Amy的妹纸空间，居然是用github的传说中的女程序员!

好吧，虽然我上一个有印象的女程序员已经是2010年的事了，但我也是见过大风大浪的已婚程序员，不会因为对方是个”珍稀动物”就激动的。

只是她太多点令我觉得有趣了。

妹纸的吸引力

  • 妹纸喜欢折腾，还敢于折腾。潜水爬山这种事，认识的妹纸多是咋呼的表示有兴趣而不愿亲自去做，生怕坏了自己娇嫩的肌肤。的确也有一些‘悍将’，大多又不是办公室里混的。既能在办公室里呆得住，还能寄心于远方并勇敢去闯的妹纸，确实不太多见。

  • 妹纸很有想法。每月读书约4-5本写记录，看电影听音乐也写记录，还关心关心时政和”无用之事”，很是难得。我身边有些哥们都不一定能做到去关心国事天下事，除了技术就只有柴米油盐酱醋茶，人一旦向生活全面认输，就让我油腻到不行。作为一名妹纸，能做到这些，的确令人刮目相看。

  • 妹纸挺真实。和我一样，很随意的记录心情和感觉，偶尔做做自我总结审视和规划。即使不成系统，即使无人问津，也就这么点滴记录着。没错，文笔没那么好，也不吸引眼球，但我喜欢这种随意留下的东西才是真实的，句句雕琢字字润笔，可能很吸引点击量，却失去了一种味道。（真的极有文采出口成章的人除外）

  • 妹纸三观不错。对事情能独立批判式思考，敢于质疑，会去查证；对人能包容尊重的态度去看待；对世界有正确的认知，知道黑暗和光明并存，知道生活和工作的平衡意义，懂得敬畏和感恩，这些都是我很尊重欣赏的优秀品质，相当nice。

与妹纸的不同

  • 我不擅长精读，特别擅长信息搜集和略读。这导致我每天对信息和知识的获取和反馈思考量都很大，按照妹纸这种记录的方式，一个月可能会记录疯（例如，我每日看的文章信息大约能有50篇以上，大约花费3-4小时。读技术型文章和资料还要再花费2小时左右。其中约能有1/3令我有一定感悟，1/10左右我会很有兴趣去写些什么）。而资讯的阅读大部分是在上下班车上，午休时间，吸烟上厕所时间用手机阅读，导致大部分时候不便记录，最终干脆只能放弃记录。

  • 对于音乐电影书籍这些东西，我已经逐渐投入的更少。或者说，我在刻意压制。之前小众问过我，“三年你打算干吗？”，我回答是“做无用之事”，就是这个意思。我本身是感性的，喜欢思想的，但这些在工作中并不能给我带来太直接的收益，然而我知道这些东西是好的有有意义的，而我时间精力有限，怎么办？我给出的解决方案是，搁置，连同独立游戏的梦想一样，搁置。时间？三年-五年。所以，现阶段我只能去除感性，先做个机器人，去赚一份让自己以后能够感性的本钱。

结论

总之，她是个有趣的人，很高兴我的友链里面可以加个活人，（rect, kira, 你们要冬眠多久= =）值得有空过去转转。

另外，她的日记记录方法不错。一段一段的隔离分块整理，既不会太散乱，也不会让人做作（对，我挺不喜欢对日记做任何修饰，整理的时候像嚼一个吐过的口香糖）。

就这样。

目标与技能支点

这一直是我擅长的话题，甚至我还自造了一个“技能支点”理论。它的意思是说，“当你想达成一个目标时，想办法将自己的技能形成一个网路，互相关联并共同支撑这个目标的实现”。

15年前在大学的时候，我就有三大目标：

  • 赚足够的钱，用来承担父亲丈夫和儿子的责任，同时为自己赢取时间自由的空间。

  • 独立做一款自己能够喜欢的游戏

  • 尽可能帮助周围需要帮助的更多的人

这些目标是层层递进的。15年过去，这些目标没有任何变化，只是更加明确细节并且增加了一条了。

  • 【第一目标】赚当下600万元购买力的钱。180万用来自身养老保险，250万用来移民购房定居和小孩教育，70万用来做父母养老医疗应急，100万用来投资兴趣爱好。（一般这个值会按照年4%左右程度增加以应对通膨）

  • 【第二目标】最近增加了一项：读万卷书，行万里路。从”生存”的世界走出来，我开始追求”生活”。不想自己再是“平时工作学习，下班聚会吃饭KTV，周末逛街旅游”的过着，我想有自己真正的兴趣爱好，能达到参加比赛或是能赚钱的那种；我想有多次的深刻的融入的旅行（起码2-3个月的居住），而不是走马观花的景点照；我想多读”无用之书”，就是那种读了也不知道以后能拿来干点嘛，可能是思想，哲学，技能工具，小说，历史，不拘泥形态，但无形中它总会融入你血肉的那种。（其实，增加了这一目标，才是最近三年在国外的最大的收获。）

  • 【第三目标】自己的游戏也明确了一些。应该是MMORPG类型方向，有世界观有群英故事有一定寓意内涵，注重交互合作和自由度。

  • 【终极目标】“帮助他人”这一点没有考虑太多，更多会考虑结合自身感受，从生存基本设施层面帮助那些“努力了却连生存都艰辛的人”的区域。

于是，目标定义下来，就是支撑它需要的东西。

钱，是必然的，但它不归于我的技能支点范围内。它是一种手段，在技能之上，目标之下。用技能获得金钱，从而达成目的。我这里只讨论技能支点。

之前我也有建立技能支点。

最开始我是做客户端，服务器，渲染，网络，工具各种调岗打杂，从PC做到页游再做手游，一方面是随波逐流方便赚钱，一方面是全面了解游戏开发各模块，为做独立游戏做好准备。期间也有学美术，学着整理数值写策划案，想转行制作人，虽然未有机会，但过程中还是学到了很多。

然后因职业生涯和生活所需，学互联网前后端开发，大数据数据分析，DevOps运维知识，渗透破解等，则在进一步全面支撑程序技术的全面性。

但前一阵我发现自己做的有点过了，目标有点歪了，所以必须要拉回来。

我的目标不是要做程序大牛，能靠它生存能靠它打工赚钱就足够了。所以我已经放弃了互联网开发和大数据方面。仅留下了黑客渗透和游戏开发两个技术领域，一阴一阳，共生互补，足够了。

但是上述的技能支点都仅在计算机技术领域，这是我生存，以及实现第一第三目标的核心，但第二第四目标和它几乎无关。

于是我需要建立一套新的技能支点体系来支撑第二第四目标。

之前有讲，我学了水肺潜水，自由潜水，能和他们形成一套技能支点的就是航海。

而航海是个巨大核心点，它依赖的细节支持还很多。整理如下：

  • 一级重要支点：外语。核心是英语，其次我的强项日语，有兴趣的西班牙语和法语。

  • 二级重要支点：烹饪，简单的船舶修理

  • 三级重要支点：读书，健身，自由潜，水肺潜，程序外包，美术等

  • 附加点: 渔猎，海钓，摄影，基本自保手段（泰拳，枪械），冥想等。

  • 脑洞级附加点：滑雪，徒步，登山等

所以有意思的好玩的值得学的东西还很多，对吧，只能继续计划了。

Python算法课程笔记[未完成]

我想复习数据结构和算法，于是学的是 中国大学MOOC的数据结构于算法Python版

  • 算法复杂度：

    常数 < log(n) < sqrt(n) < n < n(log n) < n^2 < n^3 < 2^n

    好吧，我承认之前总把 O(n) 和 O(n(log n))的大小弄反。。。

  • 简单的时间复杂度统计代码如下:

    from timeit import Timer
def test():
l = list(range(1000))
t1 = Timer("test()", "from __main__ import test")
print("cost %f" % t1.timeit(number=1000))

    代码简单，不解释了。可进行相当高效的时间复杂度测试。

  • python的list就是C++的vector。

  • python自带的dict确实性能可以，核心函数都是O(1)复杂，几乎可以放心大胆的用。

  • python的list可以使用 myList[-1], myList[-2] 做反向取值。

  • python的list初始化可以用 myList[0 for _ in range(100)] 这种方式进行。

  • python的tuple是(), list是[], dict，set是{}

  • 为方面记，可以记 heap堆(dui)是队(dui)列，然后去想排队FIFO；而stack栈就是相反的，就OK了。

“现在是最好的时代，也是最坏的时代。” 忘记这句话是哪位名人说的了，在一定范围内一定程度上我是赞同的。

当前是信息时代，信息爆炸，好的是我们蹲在家里就什么都能学到，不用以前那样爬山涉水求师学艺了。所以Linux之父Linus Torvalds才会说“把我丢到衣柜，给我一台能上网的电脑，再定时丢个披萨，我就能活得很好”，3D图形之父John Carmark也会说“在信息时代，客观障碍已不复存在，所谓障碍都是主观上的。如果你想动手开发什么全新的技术，你不需要几百万美元的资金，你只需要在冰箱里放满比萨和可乐，再有一台便宜的计算机，和为之献身的决心”，（并且后者在无聊造了几年火箭之后，突然有一天回来学神经网络，仅用一周，就不用依赖库纯看paper手工撸了一个CNN……这种变态的学习力理解力……），总之，这个时代很好。

但同时，这也是最差的时代，一方面信息爆炸，导致我们需要学习了解的东西大幅度增加，学习和享受生活的平衡调剂变的更加重要，我们必须开放勤奋跟上脚步；另一方面以“娱乐至上”，“大数据信息茧”，“嘈杂信息”等都在干扰我们有意义的信息摄取，对信息的筛选能力要求更高，一不小心就会被利用。我们又要如履薄冰保持自我。

要跟上时代脚步，又要排除短期快乐，保持客观的自我，唯一的解决途径只能是“不同的全面的深度的学习”。

毕竟工作了，于是我只能选择自学。

接触了一些学习网站，各有特色，我这里记录分享一下感受。

  • khan学院：有中文站，但仅有简单的小学课程。英文站里面主要也是以普及型教育为主，不适合学习专业课程技能。以后航海时候给小家伙进行homeschool教育很适合。

  • Couresa：我最喜欢的站，但中文课程很少，有英文字幕，而且有收费的证书（可以绑定linkedIn），名牌大学课程不少，可以一边练英语听力一边学专业课程。有很多大神级教授，真的真的非常厉害。

  • edx：用过一段，和couresa基本相同，但主要问题是，经常有课程没有字幕……我这边学习条件受限，没有字幕基本无法进行学习。

  • 中国大学Mooc：中文的，授课老师也都是中国各大学的，课程不错。对我而言的缺点之一是：我真的很讨厌跟番。我的性格适合一口气去刷完某课程的学习，但很多课程未完结，强制要等两周，我这边热情一退就学不下去了。另外一个问题是，有些老师还是喜欢照本宣科，感觉没有欧美老师讲的带感。但优点是，一些复杂的没有任何底子的课程可以在这里学，毕竟是中文。例如直接从0学金融或者医疗，英文版的课程真心没办法听。

  • 网易公开课：可以学一些好玩的课程，但感觉还是太杂，不够专业，更多是一些经验类的课程，例如“30天学会photoshop，如何提高记忆力”这种，只能作为心情调剂课程来学。

  • 其他的 TED,一席，这些只能算思想交流演讲了，比微信公众号文章会深入一些，但还是不成体系系统。在我的个人定义中属于娱乐范畴，当然比抖音那种纯快感娱乐还是有意义很多的。

  • 再进一步，特殊类需求的只能找专业网站学习了，例如学潜水技巧，学帆船结构神马，黑客技巧，电路焊接的，上面网站就都不行了。

总之呢，今年还是比较务实的学了一些编程相关的专业技能课程，算是没愧对时间。接下来还是想再去couresa拿点证书，虽然证书本身没什么实际意义，只能算一种自我鼓舞奖励吧。

读书

帆船运动百科

看完了《帆船运动百科》（The sailing bible中文版），嗯，这是一本新手入门的纯技术书。看完的感受是帆船航海真是硬技术，要学的东西可真不少。现在仅看一遍大约消化只有30%；而看程序技术书籍，速读一遍下来一般可以消化60%多。航海确实知识更杂。不过，我就喜欢有挑战的杂乱的而且有意义的技术，海上这些技能就是硬通货，会则生，不会则死，实在很棒。扯远了，我接下来要精度一下RYA的课程书籍。

扬帆追梦 帆船自驾环球之旅

看完了万军的《扬帆追梦 帆船自驾环球之旅》，一位50多岁女性写的环球游记，很有意思。女性简直是天生的作家，能敏感的发现生活中的苦乐，我纵然有同样的经历也恐怕写不出什么。术业的确有擅长和专攻，希望我老婆能尽快找到自己确实有兴趣的或者擅长的点。

另外看完这本书还得到三个心得：

  1. 外国人真是强。有2000美元买个破船就环球的黑客少年；有不到20岁就单人驾船环球的妹纸（路上船还坏了，竟然就飘着…）；有一毛钱不用家人的美国高三学生自驾环球行；有用了16年从美国驾帆船才到亚洲的驾船环球夫妇（俩人出门时候62岁，现在78岁…）；还有每个无人岛上几乎都有几个现代版鲁滨逊；….总之出乎我认知的变态极多。我本以为自己做为一个程序员，敢考虑开船航行已经算有点想法有点勇气了……结果，呃，算了，还是自己视野太小，不知道那些圈子，导致太狂妄自信了。

  2. 航海中大自然比我想象的更强大。且不说那种没准备的或者故意去作死的人，作者这60岁的夫妻（她老公是很资深的帆船爱好者了，领航图和船舶维修都很专业，还有大量旅行基友）努力好好准备。他们这一路，也是各种惨，船进水了，马达坏了，帆掉了，支桅断了，没风了，断粮了，撞船了，手砸扁受伤了，暴雨飓风火山地震全部来了一轮，真心的心脏提到嗓子眼，灾难来临，好多时候什么都做不了，只能乖乖求老天爷饶自己一命。在城市呆久了，太容易盲目信任科技，还是去船上学学如何敬畏大自然的好。

  3. 船上的技能真是硬通货。看帆船航海技术书时候就觉得要知道的挺杂的，看环球航行记录更明确了这些技能的重要性。无论是求生，还是保证航程的舒适性，都依赖于自己的各种技术，对潮汐气象的了解，机械的维修等，以及各国各种语言（英法西基本必学）的沟通交流。还是需要积极做好准备才行。

看片

嗯，看了《越狱》season1-5《囚医》《自白》《神测》season1-5《Lost》season1《POI》season1的简介，基本没啥收获……我看完电影三秒就忘。好吧，尽力写点儿试试。

《越狱》这次是复习，因为之前认认真真看的一遍都忘了……（手动滑稽）总之，让我印象最深的一个是马宏，综合战力爆表，办事极其可靠，除了第二季做的太狠，其他时候都很喜欢。一个是同铺的墨西哥好兄弟（…我忘记他的名字了），总之，也是极其可靠重义气。另外印象深刻的是他们开始想逃到巴拿马，后来想逃到哥斯达黎加。这俩国家都是我先前觉得很值得移民的地方，莫非那个时候受到了这个片子的潜意识影响？另外，主角逃跑计划就是用帆船艇，更坚定了我一旦移民到加勒比海或者中南美区域一定要买帆船艇的想法。

《神测》是讲法医破案推理的。BUG有点多，小众也这么觉得，本身也就当娱乐片看看了。里面各种奇怪邪门的病症让人菊花一紧，只感觉自己活到现在竟然没啥显性奇怪病症也是一种奇迹。

《囚医》讲的也是医生，更多是讲权力斗争，比《神测》好看一些。剧情非常紧凑，感觉有点点像第一集的《越狱》，就是主角破了一个局，反面角色就给他再挖一个坑，时刻都感觉主角进了别人的套，随时要人头不保，他却总能很自信的破局。不过，剧情反转的有点太多，主角光环也有点重。而且，那个宣敏植大反派才是最大赢家吧。。。吕布也才三姓家奴，这货反水都有10次了吧……最终还各种升级。难道导演想告诉我们“做坏人也可能终有好报”这个道理吗。

《自白》也是将破案和权力斗争的。呃，没有《囚医》那么刺激那么多变态的反转，人物之间的关系也更加戏剧化的复杂，感觉是把100个人的关系硬给塞给5个人身上了，就显得不足够真实。不过，剧里面青瓦台都已经一片黑了，女法官可真是一直强硬的很，没有靠山，谁都敢办，这也是够666的。记者也是凶残的很，问议员丝毫不虚，报纸也敢印刷。韩国片子最近胆子也是真的大，青瓦台没事就被丢出来黑一圈。我在想，啥时候我们政府能像韩国政府一样“黑暗”，估计国家就真的有救了。

《Lost》脑洞太大，我跟不上。纯当娱乐剧看了，没啥感受。

《POI》被称为全季度9分神剧，只是我仅看了第一季，感觉它的节奏把握很好：既不会一下子给你塞一堆问号，也不会完全的上下文无关。像是一棵树，支线逐步的缓慢的增加，引人入胜。上面说的《神测》也是这个模式，一集会讲一个独立故事，然后穿插一点点主线，最终若干个独立故事讲完了，主线的脉络也基本清晰。这种叙事方式我是极其看好的。其实游戏也是一样，现在流行的吃鸡，MOBA都是这种方式：每一局都有独立的全新的战场，你需要重新选角色，买枪买装备，该局内部的升级装备等成长并不会带出本局；但整个角色的成长，却又确实是每一局一局胜利逐步赚到的经验和金币。这样既接近无限的增加了游戏故事的长度，剧集和游戏的寿命可以变的更长，而每一集/每一局内也有自己的趣味，不会无聊。之前的MMORPG因为游戏时间问题，需要设计一个巨大的故事线，角色成长数值也被迫增长，导致不得不设计出巨大的天文数值，要么就拆出更多资源线（角色升级完了强化装备然后升级宝石再强化坐骑再出翅膀）。但玩家确实又期望成长，怎么办？每集资源回收就好啦~哈哈

……好吧，最近KALI视频出了一些问题，看的片是有点多了。

音乐

心做し

最近在听《心做し》，跟滨崎步的《My all》中岛美嘉的《僕が死のうと思ったのは》有点相似，都有一种悲痛的感觉在里面，但感觉还是差了一级，没有MyAll的隐忍的痛感，没有僕が死のうと思ったのは的发人深思，我想，应该是心境问题吧，毕竟我是已经过了恋爱的年纪。

三寸天堂

另外因为搜索张信哲的歌，顺道被推荐了《三寸天堂》，（别问我网易推荐算法怎么搞得，我猜他们K-means聚类之前的降维肯定出了问题），这个歌我倒是熟的掉渣。作为纯直男，真正意义上看过的爱情剧，貌似只有《还珠格格》和《步步惊心》了，一个是还在懵逼暗恋的时候，一个是我初恋内心最柔软的时候。现在听着还是挺感慨的。但是歌这个东西，几乎是个最不适合推荐的东西，几乎无法分享无法感同身受，所以不想写太多的感受。人嘛，真的是生而孤独。

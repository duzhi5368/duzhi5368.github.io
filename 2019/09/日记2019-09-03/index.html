日记2019-09-03

九月 3, 2019 发布在 日记, 计划, 2019

嘛。最近比较忙，小任务挺杂的，连绵不绝。貌似之前也有过这样的时间段，总之事情很杂，开发语言也是N种轮换的切着用，不算太开心。

大约俩三个月前，我被分到架构组，公司架构组只有三个人（两周前还离职了一个，现在只有俩），直属技术副总监管理，可能因为坐这个职位后，总出席各种中高层会议的缘故，大家会觉得你很牛或者有“前途”，现在很多人来咨询各种技术问题，搞得我无法专心学习和干活，有些不胜其烦。

其实我没资格坐这个位置，并不清楚技术副总监怎么考虑的。我的主要技术栈和其他本部门的同事并不相符，也自然无法对当前主要业务进行架构设计。

不过之后的工作安排大致明确了我的实际职位：万事屋成员。

嗯，黑帽安防，ASO，驱动开发，树莓派硬件调研，网页API服务器开发，PHP扩展，游戏开发，前端JS优化轮了一遍，挺好……越来越什么都不懂。

我写这个日志是发现最近身边的人越来越酷，只有自己在原地踏步，挺惆怅的，所以写了下来。

LILY这三年不仅移民改了国籍，还买了3-4套房子，业余读完了MBA，还接了移民的中介外包。（我的移民计划还在路上……）

光光和我一起考的OW潜水证，现在已经是自由潜教练，带了三批学生，眼看就能全职去感受另外一种生活方式了（好吧，我水肺潜水倒是确实百瓶高手了，但也不是教练，而自由潜成了他的学生）

胖子当时号称要一起去开公司做游戏，现在确实成了老板在国外做游戏（虽然我们之间有了小矛盾，我没过去入职）

DAMON也跳槽另外一个公司做了技术副总监，当时我们一起做的项目他还做成了外包（当时核心技术是我做的，但项目未果，结果他出去后换了个方式就落地了，不得不服666）

之前的大数据老大GEORGE也已经在台湾做了技术总监，整天香港国外的到处跑，家里四层小楼开心到不行（我还是个small condo，老老实实蹲老坑）

算了一下，这三年了除了工资上涨，技术面扩展，买了个房子攒了点钱，貌似是一事无成（OK，好吧，后来想起来还学了点泰拳健了健身拿了潜水证这些小欢喜）。。。略有蛋疼。

我也想做点大事，大的改变，但现在看起来，只要不离职，除了英语就只有黑客这条路好走了……有点不开心。因为公司的情况是不会让我专心做一件事的，而以现在业余时间的自学能力，估计会没什么进展。

我要琢磨点能做的有意思的事情做做才行。

如果实在想不到，我就要考虑等老婆回家生孩子的时间，停薪留职去斯巴达式学英语了。

没有标志性的激励，人生好无聊。

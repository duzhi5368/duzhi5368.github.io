[计划]总结2018展望2019

十二月 10, 2018 发布在 计划, 2018

总结2018，展望2019 [55分]

2017年目标（记录在2017年12月19日）

  1. 房子全部装修完毕，存款10W。视情况尝试理财。
  2. 英语加强，日语做恢复性训练，没事看看西班牙语。英语要达到生活完全无障碍级别。
  3. 作息，饮食，锻炼，读书保持良好习惯，潜水泰拳持续保持。可视情况恢复剑道练习。
  4. WEB前后端开发学习完毕并尝试接私活进行锻炼。（ES6,TypeScript,CSS3,Boostrap,jQuery,React,Vue,Node.js,express.js,MangoDB）若有盈余时间，加强程序类/通用类英语阅读和扩大词汇量。
  5. 和老婆去坦桑尼亚玩一圈。
  6. 摄影，画画，写作业余时间加强。摄影尽量提升到一定层级。

结果：

  1. 第一目标完成。房子装修完毕，养老理财已参与（已缴纳1/5，手头结余1/10）【目标达成100%】
  2. 英语上半年有认真学流利说并且有每周三次的英语班，下半年接近荒废。日语西语浅尝辄止。英语提高还是有的，但没有达到理想状态。【目标达成40%】
  3. 潜水泰拳中间均有2-3月的闲置，丢了又捡，捡了又丢，但总结起来当前健康状况良好，腹肌还在，体重略上升，潜水70潜达到考教练需求。其他各项表现平平，技术书还是看了不少，可以认为【达标60%】。
  4. Web全栈部分因业务需求有轻度提高，但远未达到预期。【达标10%】
  5. 充分准备了塞尔维亚，但因极特殊情况没去成，但完全非我之过。中途无聊去了马来西亚转了下【达标100%】
  6. 不解释。【达标0%】

其他：

  1. 今年重新制定了技术投资方向，从游戏开发人员转职为大数据开发人员。对Golang已经达到熟悉状态，JS提高到熟悉状态。
  2. 今年自学了大量大数据，区块链和人工智能相关技术，对大数据相关领域知识已经成功入门，明年3-4月左右应该就可以以合格的大数据工程师去应聘就职了。
  3. 今年沉迷了一段游戏，找到了一些独立游戏开发的灵感和推动力，但时间开销较大。
  4. 花费了部分精力在小孩身上。

所以整体评分大约为 55 分。令人不满意。该认真一些了。

明年计划：

  1. 养老理财手头缴纳和结余到3/5 - 4/5之间(视需求拿不拿绿卡)
  2. 死磕英语雅思5.5，必须要上学习班。
  3. 仔细考虑数字游牧的方向，并做初步准备和基本行动。
  4. 潜水泰拳健身维持并考虑轻微加强，作息饮食读书维持并轻微优化。
  5. 技术方面大数据能满足入职需求。
  6. 去Svalbard一趟。

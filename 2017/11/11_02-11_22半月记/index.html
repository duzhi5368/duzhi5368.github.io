11_02-11_22半月记

十一月 22, 2017 发布在 日记, 计划, 2017

这半个月没有怎么学习和提高，但之前想法上的种种纠结思考，最终结了个果。

本月近况

家人3号过来了，月底回国。考虑到和宝宝以及父母相聚的时间并不多，这两三周没有任何自学计划，包括健身，英语也都挂起了，基本周末就在请假陪家人出去玩，平时下班就在家里和老婆聊天。

不过，并没有负罪感。

突然有点理解在老家在小城村的那些“不思进取”的朋友们，或许大家仅仅选择了不同的路罢了，就如渔夫和富翁的故事都清楚，“谁更正确一些？”。实在太难定论了。 维持这个状态到12月再说。

思想上的变化

不知道是因为小孩的原因，抑或是当前工作离游戏逻辑更远了的原因导致，我对前路更模糊也更清晰起来。

总之，与之前不同。

我的确越来越降低了用来进行技术提升所占用的时间比重，并将这部分时间用于思维的拓宽（例如移民信息，投资理财信息，语言学习等）。我想这是低头走一段就需要抬头看看的间歇时间吧？

我一直有过一段时间就更新计划和调整方向的习惯，但一直是在技术范畴内的更变，像这样战略性的调整确实是少见的。不知若干年后回望，这次转道是否正确呢。 我实在不想再用战术的勤劳去掩饰自己战略的懒惰了。

计划的方向

我现在以投资移民为第一目标准备。目标地点基本已定，之后慢慢说。

再过2-3年申请居留，过4年左右申请永居，争取在自己40之前拿到永居。永居到手，立刻筹备购房移民。

所以

  1. 获得财富作为重心，赚钱省钱理财。
  2. 其次是语言，英语和西班牙语。
  3. 再次是保持身体健康，作息饮食锻炼。
  4. 然后旅行，读书已成习惯，辅以用摄影，画画作为休息调剂。

计划中的模糊点

未来一些东西很明确，一些东西还有些模糊。

  • 移民目标国家不是精准的一个国家，而是一小片区域，虽基本环境相似，但还需要进一步探查。
  • 移民后到底依靠什么维持收入，这是对于“未来”的一个最大的模糊点。
  • 上班闲暇时自己研究的技术方向还是在犹豫，这也是对于“未来”的另外一个模糊点。

总结

这是从07年初开始，第一次非常明确的放弃以技术为第一核心，也是第一次明确将移民提为第一核心目标。

放弃技术核心，意味着之后思考层面，为人处世都会有所改变，例如就不能以前那样倚仗技术还行而说话强硬，也必须进一步放下面子和接受别人的批评，对于公司出现的一些问题也要更全局的考量，而非以个人喜恶率性而为。

这对我而言是个不小的挑战，和多年的做事风格也有一定的冲突，然而前路还有很长，调整也是适应环境和时代的。

这个想法不是突然性产生的，从3月开始定的十年计划，到中间对第二职业的多次准备，到这次的明确定义，都是一步一步潜意识在进行着的。

动静挺大，不知正确与否，只希望能做到

愿我出走半生，归来仍是少年。

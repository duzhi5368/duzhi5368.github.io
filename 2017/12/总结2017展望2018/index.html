总结2017，展望2018

十二月 19, 2017 发布在 计划, 2017

很快就要圣诞节了，然而这里没有雪。

因为成长在北方的原因，我一直对雪是有所喜爱的。小时候就不必说了，和伙伴们打雪仗，骑车冒雪去网吧充满欢乐；而我惊讶的是似乎之前每段恋情的回忆都和雪有关系，送冰回家时她给我圈上的围脖，在佳音楼下第一次喝的酒，和静在风雪里第一次相拥，在谨谨楼下苦等的一整夜，还有和老婆在利斯特维扬卡小镇夜晚逗牛……有些已经过去，有些还将继续，每一个都是深刻难忘的回忆。

千岛之国没有雪。然而，时间却是相同的又过了一年。

本想明年初再做总结的，但想到接下来圣诞要去潜水，假期还要给新房买各种家具估计没什么时间，还是提前进行下吧。

2016年目标（记录在2016年1月3日）

  1. 自己存10W。攒钱是明年第一目标。
  2. 英语能力达到雅思4项5.5左右。
  3. 一款主导的独立游戏，完成度80%以上。
  4. 继续保持运动和锻炼，有一两种热衷的进入中级的运动方式。
  5. 适当进行进一步的作息规律调整，尝试养成一种惯性看书的习惯。

结果：

  1. 第一目标完成。房子+装修+父母过来消费+存款。本项达标100%
  2. 英语持续性一直在学，平均每天1个小时是足够的，中间仅特殊原因断了一个月，但精力还是不足够。5.5显然是没有的。但基于鼓励原则，可以认为达标60%。
  3. 今年完全没做独立游戏相关事宜，只在C，#MonoGame,VoxelGame引擎折腾了一段，只能算达成20%。
  4. 保持运动和锻炼，是坚持了的。虽然泰拳和潜水没有进入中级，但持续性很好。健身中间停滞了一段时间，还有待加强。本项可以认为达到80%。
  5. 作息规律性现在很好，也有了惯性看书的习惯，今年看了40+本电子书，4-5本纸质书。这项也可以算达成80%。

其他：

  1. 今年非常明确了移民的目的，方式，和规划，以及第二职业，被动收入的部分细节。
  2. 今年学了一些摄影理论和知识，博客进行了迁移，简单回顾了一点点美术。
  3. 今年C#以及MonoGame部分还是学到了不少，另外开动了WEB全面开发学习。

所以整体评分可以算的上有 80 分。相当不错的成绩，不完美，但是很满意。

明年计划：

  1. 房子全部装修完毕，存款10W。视情况尝试理财。
  2. 英语加强，日语做恢复性训练，没事看看西班牙语。英语要达到生活完全无障碍级别。
  3. 作息，饮食，锻炼，读书保持良好习惯，潜水泰拳持续保持。可视情况恢复剑道练习。
  4. WEB前后端开发学习完毕并尝试接私活进行锻炼。若有盈余时间，加强程序类/通用类英语阅读和扩大词汇量。
  5. 和老婆去坦桑尼亚玩一圈。
  6. 摄影，画画，写作业余时间加强。摄影尽量提升到一定层级。

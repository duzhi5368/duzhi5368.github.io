2009年3月日志

20090319

一边用Ogre写公司客户端代码( 预计9月份上市,而至今依旧连基本的Player类还未完成的客户端,公司小气不愿开薪,只能招些新手,基本上无济于事,客户端全部要自己搞定 )

一边用Irrlicht写外包项目代码( 5月份就要交)

一边用Gamebryo2.5写中间层代码 ( 预计5月左右出Demo以拉投资 )

最要命的是….现在和两个女生住一起,要帮她们洗碗,要陪着买菜….另外还有位未成熟起来的老婆….

简直想找个地方去长眠….

动漫已经四个月没看了,游戏除了中午和同事Dota一个小时以外,也基本没有任何时间了….我快脱离之前的生活了吧.

烟从三天一包变成了一天一包,喝啤酒也几乎成了习惯,Oh MyGod.

因为有了女朋友,还想着去努力学习去生活,却发现自己天生不是这块料….看着那两位翘着腿趟在沙发上看<快乐大本营>,看一个又一个港台搞笑片,我死活无法提起兴趣

做饭更是快成我深恶痛绝的事情了…..我真失败- - 自己比以前更加讨厌生活了

呼,不说这些了,上周日抽了半天时间做红娘,带着另外一对到中山公园溜达了一下,以下是唯一的一张照片….发现我笑的好邪恶…

哎,哥们,告诉我,接下来该怎么做…..SHIT

程序兄弟们很快能有点收获了,我近期会把自己的部分代码放到网上,以供大家参考交流.

补:

昨天一个Sega的制作人联系到了我,一起谈了下游戏开发的情况.

他介绍开发流程的时候,人家一款游戏,先是 绘画版,故事版,镜头设计,动作捕捉,视觉设计,音效取样 做一堆之后才开始真正的程序编程….而中国,一切都省了,连制作人这个位置也直接砍掉….程序一上来就是直接编码…..我和那位一说,那哥们直接傻了,说”你们中国的程序员真的很 si ko i (牛B)“,我无奈,只能干笑…..

嗯,日语不错的游戏程序可以联系我..可以内部推荐.在日本,内部推荐成功几率很大的喔~: )

另外,之前的项目总监大兵也跑去越南了….哎,就又留我一个人傻傻的呆在国内了….

再睡三年吧.

七月 30, 2024 发布在 日记, 2024

这半个月大致就是继续在调整方向，其根本原因还是去欧洲的计划给逼的。

虽然在美洲躺平是可以的，但从被动收入的角度来说，有自己的事业并增加被动收入的本金也是应当去做的事情。而一旦要去欧洲，我依然要去做一些事情获取更多的收入，所以最近在调研这些事情。

20240718周四

今天全面整理了一下日志，将先前一直犹豫的问题解决了，承认了个人独立游戏的失败，结合最近的信息和思考重新规划了一下接下来一两年的时间投资方向，还是很费神的。

接下来重心为以full stack web develop技术栈去找Remote job，所以要全面恢复程序员心态，同时加强英语能力。

20240719周五

今天重新整理了LinkedIn的简历，之前的英文是真的不行，整理的时候简直觉得不堪入目，想不到就这还有老外联系，真是辛苦那些HR了。

开始进行全栈学习了，包括HTML, CSS, Es6+, js, React, NodeJs, SQL, NoSQL，REST API，DevOps全面入门。总共30小时，月底应该能学完，大部分已经都非常熟悉了，算是热身复习，挺好。下个月可以搞专项了。

CSS for Programmmers 第一章学完了，共7章节，估计2，3天能学完。

20240720周六

之后需要在kangregate, itch, Desura 看看小游戏，做一些游戏分析，同时需要对其他的独立游戏收入数据做一些整理了，之前还是整理的不足。

CSS for Programmmers 第二，三章学完了，感觉干劲不足，学的比较慢。CSS这东西比较考验记性，不要求太多项目设计，所以带来创造性的乐趣不多；还好，也不算讨厌。可能是纯英文学习，多少有一点累。

整理了数个独立游戏人员的开发周期和收益，真心感觉还是能搞。之前的预期太高了，的确是身上带了太多商业游戏的气息，品味太高了。蛮多独立游戏开发的品质不高，却能奇迹般的活下来，收益还很不错，出乎我的意料。

20240721周日

今天继续整理独立游戏开发人员的周期和收益，看起来还要整理3天左右，Orz。希望得到的是一个不错的结果。

20240723周二

已经整理了60套数据了，看起来情况还行，明天再把国内的数据整理20个左右，后天再在Youtube上找一下最近1-2年的数据，就可以写总结了。

20240725周四

整理了70个独立游戏开发者收入数据，在写总结的时候发现必须对Youtube做一些调研了，于是整理了一些Youtube做独立游戏开发相关视频的信息数据。

接下来需要做的还有 - 继续整理一下Youtube的收益相关信息【DONE】 - 设计一个Logo，用于Youtube，Twitter, Instagram等。 - 在Github上的网站需要进行一次全面的更新 - 看完Bilibili上的独立开发系列视频 - 学习如何录制/剪辑视频，以及编写脚本【DONE】

20240726周五

做了一下规划的整理，学习了DaVinci视频剪辑，看了AI生成音乐音效的效果，了解了Youtue视频制作的原理和流程。

20240727周六

今天继续整理一下空间，全面清理一下。

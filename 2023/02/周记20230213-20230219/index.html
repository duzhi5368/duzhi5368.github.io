进度本周总结

本周除了晚上稍微玩了一些google play的游戏打发时间之外，感觉自己还是很尽力的投入学习了。

原感觉是每天2小时英语流利说，1个半小时多邻国，1小时youtube视频，然后晚上学了2-3小时Unity，起码也是每天学习6，7个小时的样子。

结果却是每天1个小时15分钟英语流利说；多邻国算4分钟15分计算的话，这星期每天大约1小时10分钟；而youtube视频大约每天40分钟；Unity课程是12小时，均每天1个半小时。即每日学习大约4个半小时。比预期少了1个半小时至2个小时。

而且按照这个进度的话，目标是无法达成的。

现在能想到的唯一问题，只能是效率和注意力不达标了,所以导致看似花费时间足够,但产出不足. 所以接下来,要调整作息到规范时间才行.

另外因为中间调整了一些软件APP以及学习视频，所以下面要重新整理一下预估进度和期望目标。

计划细节

4月初一个检查节点，8月初进行一次全面检查节点并进行时间调整。

  • 程序部分【每日2h】

    • Game Design and Development 2，3，4，5 累计需要55小时左右，还有不少作业，估计起码要15小时。

    • Unity base programming 需要3个小时左右，有少量作业。

    • 这俩大约需要75小时，五周完成的话，则需要每天投入2小时左右。在3月底前完成。属于核心中的核心任务。这个完成之后，重心将为小游戏原型搭建测试，争取每俩周一类小游戏，5月需要搬家会比较忙一个月，则到8月初之前可以完成大约6个游戏类型的原型。

    • 8月之后投资到unreal5和AI上面。

  • 英语部分【每日2h -> 1.5h】

    • 每天依然1个小时-2小时的英语流利说，可以保证每天一课，现在是5-13，到5-34是需要3周，即3月底完成。
    • Lv6有21课，Lv7有15课，Lv8有15课，考虑到后面课程比较难。每一个lv均用一个月，即6月底应当可以刷完，然后用1个月回顾整理，7月底截至。
    • 8月之后刷新概念英语，2册86课100小时，3册60课60小时，每天依然保持1.5小时，大约年底可学完，中间看情况可以背诵课文，以正确稳定保证雅思6.5分甚至追7分。

  • 西语部分【每日1h -> 0.5h】

    • 走遍西班牙初级上114课30小时，现代西班牙语第一册167课55小时，走遍西班牙初级下114课25小时，现代西班牙语第二册121课45小时。
    • 每天一小时，可以在7月底正好全部看完，但显然会严重消化不良。
    • 8月之后重刷的同时，带memrise, lingo进行单词复习，配合bosuu等软件一起。年底可以到A2以上。

  • 日语部分【每日1h -> 0.5h】

    • 大家的日本语初级上下大约230课3600分钟，标日初级上下大约240课4800分钟，N5+N4文法大约240课1800分钟，一起170小时左右。
    • 每天一小时，大约可以7月底可以达成日语新N4级别，即恢复到之前的N3了。
    • 8月之后可以继续重刷，带memrise, lingo进行单词复习巩固。

  • 多邻国温习【每日1h】

    • 每天还是英日西的慢慢刷即可，按现在习惯，每天1小时还是有的。

  • freelancer, upwork， 独立游戏， 资源销售【0h -> 2h】

    • 8月份之前暂不做任何考虑，之后独立游戏以itch->google play->steam路线进行测试。
    • unity asset store是个不错的平台，itch，unreal也有游戏资源交易，加上upwork, freelancer, artstation 等接外包以及宣传，顺便做些youtube视频，或许也是很不错的路子，剩余的资源自己还能做独立游戏。 就是要把美术拉起来，并且把方向从原画插画调整为游戏素材资源开发。
    • 在语音配音，音乐，原画，甚至部分剧本部分要全面考虑AI支持和研究。有空了解一下stable diffusion配合DreamBooth训练自定义风格的开销，google云和AWS的价格比较后自己搭建AI painting服务器。
    • freelancer方式优先考虑unity, unreal, go，吃饭技能，安全保本。（今年暂时不处理后面的两项）然后可以oscp, osep, oswe, osed渗透黑客为主方向，有市场有兴趣有天赋，要求必须先拿到证书。最次考虑才是网站全栈开发，资料多上手快需求多，但没兴趣竞争大

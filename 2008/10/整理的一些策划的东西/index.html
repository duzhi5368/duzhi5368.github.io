整理的一些策划的东西

十月 29, 2008 发布在 游戏开发, 2008

哈,很喜欢D&D,但是也没有十分系统的整理和学习过，更多的是靠自己玩游戏的直觉吧。所以，写了下面这些东西，勉强算是怪物卡？嘛，慢慢来，之后对武器防具，编年史，人物传这些慢慢补充吧。

呵呵，现在是把故事丢在一个不存在的世界上的，算是魔幻？反正还是受《罗德斯岛战记》影响比较多。本来想弄个大杂烩的，把中国武侠仙佛儒，日本战国阴阳忍者武士这些东西都丢进去的，想想还是算了，不伦不类是其次，主要是没时间慢慢研究了。不过之后可能还加上这些奇怪的东西进去吧~哈，受的影响比较多。反正也算不上正统的剑与魔法故事……嘛，要对亚瑟王时代的故事或者希腊神话什么的多了解些多好……好喜欢Saber的说（不过现在发现《空之境界》里的式也有点Saber味道的样子）……不废话了，有空慢慢补，本着完全不负责的态度来写，呵呵，连基本属性规则都没定……真是乱搞啊~休息了。

哥布林：

是一种行动敏锐，本性邪恶的智慧型生物。他们只是一些邪恶的，胆小如鼠的丑陋地精的近亲。他们的唯一爱好是欺负比他们更弱小的生物，例如落单的半身人和其他一些低智商单居生物。他们偶尔会攻击人类偏僻的村庄，以掠夺财宝和杀戮手无寸铁的农妇儿童为乐。他们通常以血缘关系组成小的部落，居住在别的生物的洞穴中，虽然居住在黑暗中，他们的视觉却不如想象中的那么出色，在远程战斗中他们几乎发挥不了什么作用，可依靠着敏锐的听觉以及灵巧的双脚，他们通常能够很顺利的逃离战场敌人的追杀。他们是混乱邪恶的，当一只哥布林被俘虏后，他们会毫不犹豫的出卖自己的亲人朋友以换取自己的姓名或者财富。因为怕光的关系，他们经常在黑夜中才进行攻击，在战斗胜利后，由强壮的哥布林首长分配战利品，他们还喜欢戴着帽子以遮盖自己的秃头。据说哥布林种族中有一种黄金哥布林喜欢将抢来的财宝铸造成黄金帽子带在头上，另一种则喜欢胜利后将自己的帽子沾满受害人的血以满足自己抽曲的心灵，人们通常更惧怕这种红帽哥布林。

僵尸：

它是复活的人类或其他动物的尸体，本身已没有灵魂，是被操纵者注入自己的意志而进行行动。他们行动缓慢，在操纵者意识下攻击目标。他们不惧怕疼痛，没有任何智慧，在操纵者不再进行意志的注入之后，他们将按照自己的原始本能攻击和破坏任何东西。与骷髅不同的是，他们通常是由下阶术士操纵，对命令的服从性比骷髅差出很多，而且也没有骷髅的智慧。同时，他们不象骷髅一样惧怕太阳，在持续的光照下，他们依然能够存在很久，圣系魔法对它的效果也相对比较低。

骷髅：

骷髅的出现通常是一个灾难来临的信号，他们对操纵者的命令绝对服从以及强大的组合再生能力使得他们成为不死系中最合适的战士角色。由于骷髅弱小的身体充满缝隙，弓箭类攻击几乎对它完全不产生任何伤害，他们无论从敏捷性上还是智慧上都远远强于僵尸，即使一个落单的骷髅也是一种很可怕的存在。被召唤复活之后，他们能够较好的保存还是生物时的格斗技巧和武器熟练，但它们的智慧仅受施法者的影响，所以通常来说，施法操纵者更愿意召唤一些战士的骷髅，让他们拿着剑斧盾牌去破坏。然而骷髅有致命的弱点，它们在地狱的深渊已经度过足够久远的岁月，对光照的恐惧程度更胜过吸血鬼，一旦遇到光照，一个骷髅大军将会瞬间华为飞烟，对圣系魔法抵御能力也非常之差。

独角兽：

它们是一种非常纯洁，善良，敏感的魔法生物。由于它们的角有着强大的魔法力量，通常有很多法师会雇佣猎人去森林猎取杀死它们并带回它们的角，这种纯洁的生物对人类充满了不信任，但爱好和平的天性使它们只能逃避到森林的更深处以逃避人类的猎杀。它们和精灵的关系非常友好，偶尔人类的女性和矮人也会受到它们的信任。由于长期在森林中居住的原因，它们的行动极其敏捷，而天生的魔法角也赋予了它们极好的魔法攻击和魔法抵抗能力。这种生物对战争以及不死系的厌恶是根深蒂固的，它们会不惜一切代价的赶走带来战争种子的敌人。

地狱犬：

它们原本是地狱深处的监狱守卫，拥有强壮的牙齿和锋利的爪子。它们还拥有三只头，其中一只拥有强大的再生能力，当另外两只头被砍掉后，它们能够快速的恢复，这使得很多勇者大意死在他们利爪或牙齿之下。地狱犬有天生的狂暴技能，在战斗中也绝不撤退，它们对危险的察觉本能连敏锐的黑暗精灵也自叹不如，它们憎恨秩序，害怕阳光和强大的法力。但由于召唤出来，它们通常也不接受召唤者的意志，并不受亡灵法师或黑暗法师的喜爱，而与它们签下的契约代价通常却是巨大的。

吸血蝙蝠：

这是蝙蝠成员中的一种，有人们说它们是吸血鬼的祖宗，这一传言已经被吸血鬼严正否决了。其实它们只是一种群居的食肉性动物，通常居住在山岳或海边礁石的自然洞穴中，它们高速的移动力以及强大的声纳系统使得他们成为最好的侦察员之一。虽然它们象针一样的利爪无法突破人类战士的铠甲，对手无寸铁的法师和平民还是有很强的杀伤力，得罪了一群吸血蝙蝠，结果通常是很糟糕的。由于它们通常无声无息的居住在深的洞穴中，对于喜欢洞穴探险的勇者来说，它们也是一个麻烦的敌人。

半人羊：

这是一种以懒惰著称的生物。他们喜欢沉溺于酒水和女色之中，在村庄中男性出去打猎的时候，它们就成群结队的来到村庄威胁妇女，掠夺麦酒，然后肆意狂欢，偶尔也会有些可怜的孤独旅客成为它们的目标。它们喜欢拿着钉头锤砸进生物脑袋的感觉，他们是天生的语言能力者，熟练的运用多种语言，遗憾的是，这种天赋并没有使它们成为优秀的商人，懒惰的天性使得它们更喜欢不劳而获。在战斗中，它们擅长嘲骂羞辱它的对手，然后逃之夭夭，这种怯懦的行为已经成为半人羊在整个大陆上的象征。

故事希望将正义和邪恶做为一种表面的矛盾。而混乱和秩序才是内在的矛盾。秩序是努力将世界推向一个和平统一的静止状态，而混乱是希望将一切成形的东西破坏重新制造一片无序。

基本法术属性：光（圣），暗，无，风，火，土，水，雷

基本地形包括：水，沙漠，农田，森林，草原，沼泽，山脉，丘陵，融岩，道路，冰原，空。

基本属性：

力量（物理攻击力，少量命中）

敏捷（闪避能力，攻击频率）

灵巧（闪避能力，命中力，移动力）

智慧（MP上限，魔法攻击力）

体魄（物理防御力，HP，部分状态抵抗力）

精神（MP恢复速度，魔法防御力，部分状态抵抗力）

重要类人种族：

人族：43%

矮人族：8%

精灵族：10%

黑暗精灵族：6%

兽人族：19%

地精族：4%

半身人族：2%

巨人族：1%

其他种族：7%

怪物种类：

动物系：就是我们通常在现实生活中了解或知道的各种生物总和。通常来说他们都拥有基本的智力，不会使用法术，依靠先天的攻击本能进行战斗，大部分属性依照现实中人们所记载的资料为鉴设计，但不排除会进行部分的虚构对象出现。例如：狗熊，鹰，恐龙……

不死系：通常是由各种活着的生物，死后被某种力量予以活化，通常来说，它们的智能较低，由于它们通常不属于任何属性，对非圣系的魔法抵御能力较强，对物理伤害通常也能够很好的抵御。例如：幽灵，僵尸，吸血鬼

魔兽系：通常是指天生拥有魔法能力的动物。它们和动物系非常类似，只是通常都拥有超自然的魔法特异能力，它们允许拥有较高的智慧，不过普遍来说，还是比人形系的生物智慧稍低一些。相对来说，他们出现在西方神话较多一些。例如：独角兽

人形系：这些生物通常来说智能相对比较高，有自己种群的固定文化，通常也具备了自己的语言，种群数量也相对较多。它们比较大的特征是与人类相似，有一个脑袋两只手两只脚，拥有自己的灵魂。例如：巨人，半人马，矮人，精灵

精怪系：这类生物通常是由非生物转化而来，在自然的造就之下，受到不明力量的影响和波动用有了自己的灵魂。通常这种生物拥有着一定的自然魔法能力，能够与大自然进行良好的沟通。例如：树精，黏土怪

恶魔系：这类生物在现实中只会存在于一些神话传说中，拥有着强大的力量，它们是由异次元的来访者，每次来临都将为这块大地带来绝对的灾难。它们喜好邪恶和破坏，这种喜好已经深入本质，任何一个恶魔系的生物都是绝对仇恨和平的。例如：炎魔

神灵系：这类生物也拥有着绝对的力量，它们通常在某一领域拥有着绝对权力。人们通过拜祭它们获得垂青来得到一些特殊的能力，和恶魔系一样，它们是不可被毁灭的，由于是异世界的存在，他们在这块大陆上是没有身体的，若要现身于世，必须找到适合容器才可。例如：战神，自然之神

机械系：这是一种非生物，是由类人系创造出来的物体。它们不存在自己的灵魂，只会按照设计者的预先设计进行机械的行为，该类物体不接受任何光暗属性魔法，可以由创造者进行修复。通常也会有高智能生物对其进行操控，这种操控依旧倚赖的是器械，一旦操纵器械破坏，该物体将不可进行任何行动，直到下一个创造者对其进行修复。例如：遥控矿工，投石车

伤害类型：物理内伤（几率淤血，减HP上限），物理外伤（几率出血，减HP）

武器类型：

极近程物理：徒手，指套，匕首，格斗爪

近程物理：短锤，短刀，短剑，单手斧

普通距离物理：长剑，细剑，大刀，双手锤，狼牙棒

稍远距离物理：双手大剑，枪，镰刀，叉

中程物理：飞锤，锁链，暗器

远程物理：投石器，弓，弩

在设计中间自己和自己出现了很多矛盾，例如恶魔系原本设计为妖魔系的，想把中国的七界论丢进来，后来发现 精，怪，妖，仙 这些都与西方的设定有太大冲突，很难磨合，除非设计为两个或者更多大陆，但是这样做很容易把自己陷入一个递归死循环中，于是放弃了。希腊神话中的神也打算全部放弃掉，但是太接近D&D的话实在对我挑战过大。东方的仙术想融合到西方魔法中也的确很困难，包括在设计类型的时候，很想加入大航海时代的交易和城镇自成长系统，后来想想，这样的话将针对用户群不明确，不是好的方法，也忍痛删掉了。武器方面还是采用《暗黑》和《洛奇》的风格，给予武器随机的前后缀名的好，例如 食人魔猎杀者的 坚韧之 短斧。武器技能本身有自己的熟练，武器可以增加和更换附加物品，如 斧头的 轻木斧柄，争取一个武器给予三个可提升的部位，最高+3左右，以替代宝石镶嵌。龙一直想单独做一个种族系，但后来想想，那玩意终究是要做BOSS出场的，铁定不能多了，没打算写侏罗纪战记，所以给合并到魔兽系去了，我想，真要有D&D粉丝来了肯定要砸我的场子了。

对于语言能力，攀爬能力，游泳能力，夜视能力，沟通能力，价格获知能力这些多少带了些生活性质或与激烈战斗紧密度不足的能力想暂时放弃，恐怕太复杂了自己承受不住 0。0

在这块大陆上，有一个传说，有个极其邪恶的暗黑神被封印在地下，每隔千年就会复苏，为这片大地带来毁灭的灾难。 人们在三十多年前的黎明之战中，千余名勇士进入暗黑洞穴，最终只有七位勇者到达了底层，在封印了黑暗神以及诸多副神之后，大陆度过了三十多年的稳定平和的日子。 某日，人类四勇者之一，悠久之贤者先后释放了欲望之神，死亡审判之神，疑惑之神的封印，而随后英雄王也带佣兵团到了极北之冰封之地，以那为根据地向大陆的古魔法大国发起了总攻，人们传说他是受到了破坏之魔王（即破坏之神）的侵蚀失去了心智。三勇者的最后一位骑士王发动了圣战进行对抗。 战争起始，过习惯了平和日子的骑士军团完全不堪一击，骑士王发布全民紧急招募令，无论身世的招募将才。 我们的主人公则是在此时成长的，故事也是此时开始。 之后的战争中，骑士王首先死亡，临死时他说出了英雄王和大贤者的行动原因，但由于下属A的叛变，他的遗言未能顺利传播出去。随后在A的带领下，英雄王被愤怒的民众杀死。其属下B继承遗志。 之后大贤者预估了自己将被A追杀，临死之前留下了XX预言，自行了断。 主角在某种机缘下拿到了大贤者的预言书，知道了一切的真相，他一方面努力凑齐各大水晶以维护暗黑神的封印，一方面与A和B进行着周旋。 最终主角封印了暗黑神，自己归隐去了。

人类四勇者： 骑士王：圣骑士，居住XX神圣王国被人民拥护为国王。虽然得到贤者的通知，却依旧希望使用平和的方法来唤醒沉睡的人们，最终战死于不动的桥墩。 英雄王：狂战士，带领手下和部队居住在沙尘之彼方的雇佣兵竞技场。本能发现了世界不正常的邪恶气息，通过访问悠久之贤者得知即将来临的灾难，因不屑人类的懒惰安逸，决定以自己之力唤醒大陆中昏沉的人们。 悠久之贤者：法师，居住在大陆东方接近海边的俯视万物之塔中。原收徒，后独居，一直在研究《黑之预言书》，首先发现暗黑破坏神之苏醒迹象。 沉思之修女：牧师，居住在大陆北方的沉思的圣堂中。作为修女长侍奉着神，十分厌恶战争。因此，对英雄王的行为十分反感，一直组织民众对英雄王进行着抵抗。 其他族的勇者： 风之引领者：黑暗精灵游侠。它虽为勇者之一，但由于黑暗的令人类以及其他众多种族并不对其表示尊敬。长期游荡于大陆各地。 森林守望者：精灵弓手。 山丘之王：铜须。矮人国盾战士。矮人国自卫团团长

王国体系包括：

1：神权国家：国家首脑就是教皇，即宗教首脑。这样的国家通常采取封闭外交政策。

2：议会制国家：国家首脑由议会选举，更多的代表商人阶级利益。这样的国家外交力经济能力都很强大，但战斗力通常较为薄弱。

3：帝国制国家：国家首脑是皇帝，皇帝拥有最高决策权和军队的绝对领导权。相对来说，这种国家的经济较为落后，但军事实力较为强大。

4：联合公国：这种国家通常分为大量的区域，这些区域由有战功或王族血缘关系的人领导，区域有相当的独立自主权，在经济法律上允许独立，但在军事外交方面依旧受公国国王一人负责。这类国家通常内乱较多，经济上参差不齐，军事实力较为强大，但弱于帝国制国家。

四奇： 冬之暗杀者，据说是无情，高傲，来去无踪从不与人交谈的刺客。然而在某种原因下，视主角为挚友。冰霜的表面之下蕴涵的是炽热的感情，为保护主角而亡。（人没有希望孤独的，支起屏壁只是为了保护自己而已。） 春之治愈者：看起来纯朴善良的祭祀，实际却是温柔乡的背后老板娘。（人们需要偶像，那么我就给予他们偶像好啦。） 夏之咆哮者：完全以自我为中心的狂暴战士，游荡于大陆各地（我不懂所谓的信仰，我的信念就是恶即斩。） 秋之隐匿者：有人说他是术士，有人说他是游荡者，其实际身份无人知晓。实际是藏匿在宫廷之中的智者，在最终战役前做为军师战亡。（小隐隐于林野，中隐隐于闹市，大隐隐于宫廷）

故事段2： A原本是因品德问题被废黜的罪之骑士之一，后不愿在东方矿场服役自省，越狱而出，成为游荡者。 骑士王生前有几为骑士团长，其中首位骑士受到人民最多的赞喻，被尊为“秩序之骑士”。但由于他长期带着面具，除了骑士王本人外，其他人均未见过他长相，在骑士王死亡前，他刚将A追捕到，在审问A的时候，骑士王的刺杀者被秩序之骑士发现，战斗中秩序之骑士死亡，被A无意中取得其面具，在骑士王死亡后坐上了国王的宝座。在其成为国王之后，虽然行为不大符合骑士原则，但国家国力明显增强，取得了几次战役的胜利，受到人民的拥戴，但最终被持有魔剑的B揭漏了真相，在人民责骂唾弃中被送上绞刑台死亡。 意义：// 战争时应当有战争时的秩序和规则，根据情况变通的秩序才是最佳的。 // 人们信仰的通常仅是一个符号，在符号背后到底是怎样的面孔，无人知晓。 // 有过错的人应当有自己的权利去选择赎罪的方式。 // 有过错不代表没有才能。 // 人类单纯的害怕欺骗，习惯迅速的将责任转给他人，对恩情却忘记的极快。 留有问题有：1：因什么品德问题被废黜。 2：应在A为游荡者时增加与主角的接触，以增强它对主角的影响。

故事段3： 主角成为圆桌骑士团之一后，发现圆桌骑士中有好几位年迈老者，战斗中他们贪生怕死，长期主张与雇佣兵议和给予礼金以保安稳。在一次战役后，原本艰辛取得的胜利被他们拿去做为了议和的条件，归国都后，人民欢呼着迎接这些老骑士们，令主角异常不满，后经一位成熟的中年骑士引导之下，认识到是自身道德修养依旧不足，后努力加强骑士精神修炼……讽刺的是，当同样的事情发生在这位中年骑士身上后，他立刻投靠了雇佣兵军队。在一次战场中，他无意被主角所杀。 意义：// 政权和思想一样是会老化的，优秀的人也未必一直优秀。 // 铁和血的战争只能同样用铁和血去平抚。 // 圆桌意味着平等，同样可能是无能者的避难地。 // 无论战争中还是战争后，宣传媒体永远为部分人利用，人民的眼睛耳朵都极易受到欺骗。 // 事情不发生在自己身上，永远能够轻易的解决。 // 年轻人不应当完全的遵从年纪的经验，更应当相信自己的头脑

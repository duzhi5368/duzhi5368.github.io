20080908

FK_Texture测试完毕:) 日语的复习还算顺利.

北京今天下了雨,突然感觉到了秋意.我对北京的冬天有很好的怀念,两年前,薪资还不高的时候,冬天冻的可怜也舍不得买被子,拿着从家里带的军大衣包在身上,很温暖. 呵呵,军大衣的确是个好东西.可惜现在北京要买到那么纯正结实又暖和的军大衣蛮难.

秋意一来,人容易感伤.嘛,做程序这行久了,人感情退化,也更没了大学时的”文采”,写的东西朴实无聊起来.哈,悲惨的进入”程序员”意境了呢.

拿起手机,30来个号码只有两个是大学同学的,简直是讽刺,都被遗忘到哪儿了.我,真是个薄情的家伙呢~哈哈哈哈哈..

联系到了楠,大学时候的好哥们,一起暑假留在武汉不回去,坐在凉席上,一边向身上泼凉水一边玩游戏的家伙,后来寒假也是没回去,留着玩游戏,哈哈哈哈,真疯狂的以前: )

那哥们现在在老家村镇里做个什么村主任之类的公务员差事,本以为公务员会很爽,老了有国家养,不过现在看起来也不怎么样,居然还有合同期?!- -明年他的合同期就到了,然后,自己找工作….我靠,铁饭碗也不铁了,世道还真不好过呢.而且问起现在上班时候干什么,告诉我的结果是”上班就打牌了”…残念.真是糟糕的答复.很怀疑他明年该怎么办.

之后和另一个楼上的朋友峥见面了,大学时候蛮瘦的,和我有一拼的家伙,现在脸上胖了一圈,肚子也出了大将军肚,见面差点不敢认- -他现在做市场,也是大学毕业后就开始做的,一直没跳过,薪资现在也才5000.准备转行….26了转行…而且还仅在准备阶段,没有落实去做…一面佩服他的勇敢,一面庆幸自己能够很顺利的在毕业后就找到自己的奋斗目标啊…

毕业后,大家似乎都没有我这样幸运的家伙,是老天在对我的补偿么?…呵,福兮祸之所倚啊.真要好好小心自己的将来.

想想,疆哥在武汉做开发,看他日志似乎非常不顺,负责的项目结果很糟,那么要强的人的说..

永利兄在山东也不知如何,凯亮在山西安逸的在一家药厂混着,倒也符合他的超脱无争的个性.景云做老师又转行餐饮设计,学立,陈光他们照样读研.

大家就这样走上了自己的路,分散在各地.只希望大家一切都好吧.

似乎活在幸福的笼罩下,迷失了自己一样,总觉得,自己似乎进入了另一个世界,虽然看起来似乎”白领”了,看起来”文化人”了,实际上,我讨厌这样,向上走没有错,可..不对味.丢了某样重要的东西一样,到底是什么?

真的厌烦自己了.

到底现在是什么情况?什么原因…

我仔细想了啊.没错,我到北京之后,的确工作之余的时间把握的不足,但是,我工作时候绝对比以前认真多了,进步也非常快,我有自信这么说.那么,工作之余的自习也近来调整的不错啊.到底什么令我不安???!

工作?我根本不用担心工作问题啊?爱情?我没有迫切需要到那种地步.其他方面,我做的也可以啊.自习看代码编代码,人际和同事处的也很好,身体也健康,到底我在害怕什么,不安什么,烦恼什么?我应该很快乐才对啊.

见鬼…

压力大?不,我玩的也不少.每周也总会抽个时间出去溜达的,前几周不是还和道馆的朋友游泳吃饭什么的么.十一还和朋友安排了去青岛旅游呢.

想知道以前的哥们他们现在心里怎么想的.会不会比我更焦虑?还是能很理智的调整?

嘛,自己一直属于情商不高的那种,现在还不错了,与人接触的时候能顺利调整自己的心态和脾气了,实际上,我还能有自信的说,还算不被讨厌那种了,嘿嘿,不过…自己总和自己过不去,这是错误的?还是宽人严己的一个好事?

再次鄙视MS,今天写个引擎测试,CreateWindow居然一直失败!,GetLastError()错误码是183,查了一看,原因是什么”文件已经创建,不可再创建”,因为引擎本身够大,想了半天,上网又去查.绕了半天也没找到答案.( 附加一句: 网上的高手都TMD的混的,这个问题问的人不少,硬是没一个正确答案,KAO )…结果是我为了省事,回调函数简单一句 return 0; 给处理了,但是在CreateWindow的时候就会默认有一个回调消息,而该消息必须返回true,不然就会创建失败.

就这个一个简单的玩意,XD,浪费我一个小时…MS给的错误信息,封装的消息流程……都导致了这种不爽…

真的,要其他操作系统有机会和MS的Win一拼的话,一定抵制MS……再次无限鄙视McroSoft.
